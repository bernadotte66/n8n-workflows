# n8n Automation Workflows

A collection of production-ready n8n workflows for AI automation.

## Workflows

### 1. OCR AI Bot
Telegram bot that extracts text from images using OCR.Space API, cleans and formats with Groq LLM, returns structured results.

### 2. B2B Lead Scraping
Automated data extraction workflow for business leads.

### 3. Telegram AI Bot
AI-powered Telegram bot with automated responses.

## Tech Stack
n8n, OpenAI, Groq, Telegram API, OCR.Space
