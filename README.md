## Workflows

### 1. OCR AI Bot
**Telegram bot for document text extraction**

Extracts text from images via OCR.Space API, cleans and structures output using Groq LLM, and returns formatted results directly in Telegram.

- **Use case:** Automate invoice, receipt, and document processing
- **Stack:** n8n · Telegram API · OCR.Space · Groq LLM
- **Result:** Eliminates manual data entry from scanned documents

---

### 2. B2B Lead Enrichment Engine
**Automated lead research and enrichment**

Pulls raw company data and enriches it with structured business intelligence — contacts, industry tags, and qualification signals — ready for outreach.

- **Use case:** Sales teams spending hours on manual lead research
- **Stack:** n8n · OpenAI · External APIs
- **Result:** Cuts lead research time by 80%

---

### 3. Telegram AI Agent
**Conversational AI bot with automated responses**

AI-powered Telegram bot that handles natural language queries and automates responses based on configurable logic flows.

- **Use case:** Customer support automation, internal team bots
- **Stack:** n8n · OpenAI · Telegram API
- **Result:** 24/7 automated responses without human intervention
